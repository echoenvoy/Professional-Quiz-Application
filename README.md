\# Professional Quiz Application



A complete quiz application with graphical interface built in Python using Tkinter.



\## Features

\- Load questions from JSON file

\- 30-second timer per question

\- Automatic question progression when time expires

\- Score calculation

\- Detailed results review

\- Professional UI with styling

\- Progress tracking



\## Installation



1\. Make sure you have Python 3.6+ installed

2\. No additional packages required - uses only standard libraries



\## Usage



1\. Place your questions in `questions.json` file

2\. Run the application:

&nbsp;  ```bash

&nbsp;  python quiz\_app.py
